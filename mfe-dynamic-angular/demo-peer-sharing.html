How This Works:

  1. No Host Dependencies Required: The host doesn't provide Angular/Zone.js/RxJS
  2. First MFE Shares: The first Angular MFE loaded registers its dependencies in the global registry
  3. Subsequent MFEs Consume: Additional Angular MFEs use the shared dependencies
  4. Singleton Pattern: Only one instance of Angular/Zone.js/RxJS exists across all MFEs
  5. Independent Operation: Each MFE can work standalone if registry not available
